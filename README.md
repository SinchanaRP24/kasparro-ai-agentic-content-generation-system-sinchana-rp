# Kasparro AI Agentic Content Generation System

This repository contains a Java-based backend system that demonstrates agentic design principles for automated content generation.

The system uses modular agents orchestrated through a central pipeline to generate structured JSON outputs such as FAQs, product pages, and comparison data.

## Tech Stack
- Java
- Gson (JSON handling)
- Git & GitHub

## How to Run
```bash
javac Main.java
java Main
